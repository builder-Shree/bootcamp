Flex property example

It provides an efficient way to lay out, align, and distribute space among items in a container, even when their size is unknown or dynamic.

display flex makes the container flex container and its items flex.

The align items property makes the items align at main axis i.e horizontal

flex-grow: 1; allows items to grow to fill the available space.

flex-shrink: 1; allows items to shrink if necessary.

flex-basis: 100px; sets the initial size of the items.

CSS positioning allows you to control the layout and placement of elements in a web page. The main positioning properties include static, relative, absolute, fixed, and sticky

Static Position

Static positioning is the default positioning for all elements. Elements are positioned according to the normal document flow.

Relative Position

Relative positioning positions the element relative to its original position in the document flow.

Absolute Position

Absolute positioning positions the element relative to the nearest positioned ancestor or the initial containing block if no positioned ancestor exists.

Fixed positioning positions the element relative to the viewport, and it remains fixed during scrolling.

Sticky positioning toggles between relative and fixed positioning, depending on the user's scroll position. It remains relative until a specified point is reached, then it behaves like fixed positioning.
